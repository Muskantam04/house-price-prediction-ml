# 🏠 House Price Prediction (ML + React)

A machine learning project that predicts house prices based on input features, with a React frontend for user interaction.

## 🚀 Features
- Predict house prices using ML model
- User-friendly React interface
- Backend API for prediction
- Real-time input and output

## 🛠️ Tech Stack
- Python (Machine Learning)
- Scikit-learn
- Flask / FastAPI
- React.js

 ## 🔧 How to Run
